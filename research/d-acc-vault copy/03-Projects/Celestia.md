# Celestia

**Category**: Data Availability
**D/ACC Score**: 4 | **Investment Tier**: [[Tier 2 - Strong Conviction]]
**Stage**: Mainnet | **Token**: TIA

## Overview

Modular DA layer with Data Availability Sampling (DAS). 8MB blocks targeting 1GB. $155M raised. Achieved 99.9% cost reduction vs Ethereum mainnet.

## Key Differentiators

- **Data Availability Sampling** - Light nodes verify without downloading full blocks
- **99.9% cost reduction** - vs Ethereum mainnet DA
- **Modular architecture** - DA separated from execution

## Why D/ACC Aligned

DAS enables light node verification without trusting validators. This prevents the centralization that would occur if only powerful nodes could verify data availability.
