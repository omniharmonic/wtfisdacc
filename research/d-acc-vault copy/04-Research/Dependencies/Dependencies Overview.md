# Dependencies Overview

Critical infrastructure and chokepoints that D/ACC systems rely upon. Understanding dependencies helps identify:
- Single points of failure
- Centralization vectors
- Investment de-risking opportunities

## Dependency Categories

### Infrastructure Dependencies

| Dependency | Risk Level | Alternatives |
|------------|------------|--------------|
| [[Ethereum Consensus Dependency]] | High | Multi-chain, sovereign rollups |
| [[Cloud Infrastructure Dependency]] | Critical | Bare metal, geographic distribution |
| [[Semiconductor Supply Chain]] | Critical | Stockpiling (no true alternative) |
| [[Sequencer Centralization]] | Critical | Shared sequencers, based rollups |
| [[RPC Provider Concentration]] | High | Light clients, decentralized RPC |
| [[GitHub Code Hosting]] | High | Self-hosted, Radicle |
