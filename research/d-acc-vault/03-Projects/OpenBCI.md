## Overview

OpenBCI is an open-source brain-computer interface platform created by Joel Murphy and Conor Russomanno after a successful 2013 Kickstarter campaign. Based in Brooklyn, NY, it provides low-cost biometric amplifiers for measuring brain (EEG), muscle (EMG), and heart (EKG) electrical activity.

## Hardware

- **OpenBCI 32-bit board**: Uses Texas Instruments ADS1299 chip for biopotential measurements
- **Ultracortex Mark IV**: 3D-printable headset using 10-20 electrode placement system
- **Galea** (2024): Next-gen device integrating EEG, EMG, EDA, PPG, and eye-tracking in single headset

## Software

All open-source:
- Processing-based GUI application
- NodeJS and Python display/processing software
- Fully documented APIs

## Why Open Source BCI Matters

Traditional BCI development:
- Proprietary hardware (Neuralink, etc.)
- $100K+ research-grade equipment
- Closed ecosystems

OpenBCI's approach:
- ~$500-2000 for full setup
- Open hardware designs (can 3D print headset)
- Open software stack
- Community-driven development

## Galea (2024)

OpenBCI's latest product merges biometrics with mixed reality:
- First device combining EEG + EMG + EDA + PPG + eye-tracking
- Beta units shipping August 2024
- Targets research and VR/AR integration
