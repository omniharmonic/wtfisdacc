## Cross-Linking Instructions

To manually link a Project to a Sector in Notion:
1. Open the Project page
2. Find the "Sectors" relation field
3. Click to add a relation
4. Search for and select the appropriate Sector

To link a Primitive to a Sector:
1. Open the Primitive page
2. Find the "Sectors" relation field
3. Click to add a relation
4. Search for and select the appropriate Sector(s)
